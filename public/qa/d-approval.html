CEIG Drawing Approval Process

विद्युत निरीक्षक आरेख अनुमोदन प्रक्रिया

1. Objective

Submit a complete, accurate, and properly formatted application through the designated channel to minimize errors, enable fast-track processing, and get the drawing approved.

1. उद्देश्य

त्रुटियों को कम करने, फास्ट-ट्रैक प्रोसेसिंग को सक्षम करने और ड्राइंग को अनुमोदित कराने के लिए निर्धारित चैनल के माध्यम से एक पूर्ण, सटीक और उचित रूप से स्वरूपित आवेदन जमा करें।

2. Preparation Checklist / तैयारी चेकलिस्ट

  • HT Equipment List or Generator / HT उपकरण सूची या जनरेटर
  • Equipment Location Map (Safety Distances) / उपकरण स्थान मानचित्र (सुरक्षा दूरी)
  • Single Line Diagram (SLD) / सिंगल लाइन डायग्राम (SLD)
  • Equipment GA / उपकरण GA
  • Cable Route Drawing (for HT) / केबल रूट ड्राइंग (HT के लिए)
  • Earthing Layout / अर्थिंग लेआउट
  • Lightning Protection Layout / लाइटनिंग प्रोटेक्शन लेआउट
  • Protection Details / सभी सुरक्षा विवरण

Note (English): All drawings should be signed by an HT ‘A’ Class licensee contractor in Chhattisgarh. If equipment quantity is low, documents can be consolidated into one comprehensive drawing.

नोट (हिंदी): सभी आरेख छत्तीसगढ़ में एक HT 'A' श्रेणी के लाइसेंसधारी ठेकेदार द्वारा हस्ताक्षरित होने चाहिए। यदि उपकरण कम हैं, तो सभी दस्तावेजों को एक व्यापक ड्राइंग में समेकित किया जा सकता है।

3. Application Submission

Online: Submit data and upload drawing files.
Offline: Fill application and submit physical drawings to the CEIG office.

Approval can be done within 2–4 hours. No fees required.

3. आवेदन जमा करना

ऑनलाइन: डेटा जमा करें और ड्राइंग फाइलें अपलोड करें।
ऑफलाइन: आवेदन भरें और CEIG कार्यालय में भौतिक चित्र जमा करें।

अनुमोदन 2-4 घंटे के भीतर किया जा सकता है। कोई शुल्क आवश्यक नहीं है।
