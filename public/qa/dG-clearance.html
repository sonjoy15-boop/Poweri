DG Set Charging Clearance

डीजी सेट चार्जिंग क्लीयरेंस प्रक्रिया (CEA Regulation-34)

Objective / उद्देश्य

CEA Regulation-34: Inspection by Electrical Inspector is mandatory for generating units above the specified capacity before commissioning. This section ensures fast processing by complying with regulations and minimizing errors.

CEA विनियमन-34: कमीशनिंग से पहले विद्युत निरीक्षक द्वारा निरीक्षण अनिवार्य है। यह अनुभाग नियमों का पालन करके और त्रुटियों को कम करके तेजी से प्रोसेसिंग सुनिश्चित करता है।

1. Documentation Checklist

  • Layout of DG Set premises
  • Single Line Diagram (SLD) with Load/Protection
  • GA Drawing of the DG Set
  • Earthing and Lightning Protection Layout
  • Manufacturer’s Test Report & Purchase Docs

1. दस्तावेज़ चेकलिस्ट

  • डीजी सेट परिसर का लेआउट
  • लोड/प्रोटेक्शन के साथ सिंगल लाइन डायग्राम (SLD)
  • डीजी सेट का GA ड्राइंग
  • अर्थिंग और लाइटनिंग प्रोटेक्शन लेआउट
  • निर्माता की टेस्ट रिपोर्ट और खरीद दस्तावेज

Charging Clearance Process / चार्जिंग क्लीयरेंस प्रक्रिया

Step 1 Drawing Approval / ड्राइंग अनुमोदन

Submit all drawings (signed by HT A-Class Licensed Contractor) to the CEIG office for initial approval.

Step 2 Site Inspection / साइट निरीक्षण

After drawing approval, submit the following for inspection:

  • Approved Drawings / स्वीकृत चित्र
  • Manufacturer & Site Test Reports / टेस्ट रिपोर्ट
  • Earth Pit Testing Report / अर्थ पिट रिपोर्ट
  • DG Set & Energy Meter Bills / खरीद बिल
  • NABL Energy Meter Test Report / मीटर टेस्ट रिपोर्ट
  • Environmental Clearance / पर्यावरण प्रमाण पत्र
  • Format L&B and Format A / प्रारूप L&B और A
  • Inspection Fee Challan / निरीक्षण शुल्क चालान

Step 3 Charging Permission / चार्जिंग अनुमति

Submit the final application to the CEIG office along with the Site Inspection Report and all previously submitted supporting documents to obtain the final clearance.

Tips for Fast Processing

  • Ensure document accuracy.
  • Respond promptly to queries.
  • Make timely payments.
  • Most Important: Liaison effectively.

तेजी से प्रोसेसिंग के लिए टिप्स

  • दस्तावेजों की सटीकता सुनिश्चित करें।
  • प्रश्नों का तुरंत उत्तर दें।
  • समय पर भुगतान करें।
  • सबसे महत्वपूर्ण: प्रभावी संपर्क (Liaisoning) बनाए रखें।
